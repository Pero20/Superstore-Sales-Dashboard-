📊 Supersales Performance Dashboard Analysis
🔹 Overview
✅ What the Project is About
This project focuses on analyzing retail sales performance using Power BI. It provides an interactive dashboard that tracks key business metrics across regions, products, and customer segments to support data-driven decision-making.
⚠️ Business Problem
Limited visibility into overall sales performance
Difficulty identifying underperforming regions
Poor tracking of customer value and sales trends
Inefficient manual reporting processes

📌 Source of Data

📂 What it Contains
Order details (Order ID, Date, Ship Date)
Customer information (Name, Segment, Region)
Product categories and sub-categories
Sales metrics (Sales, Profit, Quantity, Discount)
Geographic data (Region, State, City)

🔹 Key Insights
📊 Sales increased by 25% in Q4, showing strong seasonal performance
📉 Certain regions underperformed, indicating need for targeted strategies
💎 High-value customers were identified, contributing significantly to revenue
🔹 Recommendations
🎯 Focus on improving underperforming regions through targeted marketing
🤝 Strengthen relationships with high-value customers via loyalty programs
📦 Promote high-performing products and review low-performing ones
📅 Leverage Q4 trends by increasing marketing and inventory during peak periods
📊 Adopt continuous dashboard monitoring for faster decision-making
